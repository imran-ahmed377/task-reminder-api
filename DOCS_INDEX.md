**Week 1: Understand the Project**
- QUICKSTART.md → Run locally
- FRONTEND.md → Understand UI
- TESTING.md → Learn endpoints

**Week 2: Learn Frontend Code**
- Study index.html
- Understand JavaScript
- Try modifying the UI

**Week 3: Learn Backend Code**
- Study main.py
- Understand routes.py
- Read auth_routes.py
- Learn database.py

**Week 4: Deploy**
- Follow DEPLOYMENT.md
- Deploy to Azure
- Monitor application

- README.md → Full overview
- PROJECT_SUMMARY.md → Technical details

**Day 4-7: Study Code**
- Frontend: index.html

- Modify code & see changes
- Test in browser & API

**Day 11-14: Deploy & Monitor**
- DEPLOYMENT.md → Deploy
- Set up monitoring
- Plan enhancements
